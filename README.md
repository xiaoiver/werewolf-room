# werewolf-room

> 一夜法官，基于socket.io和vue

## 运行步骤

``` bash
# 安装依赖
npm install

# 修改config/index.js中的proxyTable
指向后端的服务地址

# 修改config/dev.env.js中的SOCKET_HOST
例如 SOCKET_HOST: '"http://172.18.20.97"'

# 启动服务器
npm run dev

# 访问8080端口

```
